``
yum -y install ntp ntpdate //install ntpdate tool
ntpdate cn.pool.ntp.org //Set the system time to synchronize with the local network time
hwclock –systohc //write system time into hardware time

** crontab service **
``
/ sbin / service crond start (open crontab without any prompt indicating it has been opened)
/sbin/service crond status (view status)

crontab -e (Enter the i key, copy the following code ctrl + c, paste ctrl + v to join the scheduled task, you can customize and add the required functions)

``

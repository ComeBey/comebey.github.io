1. Update server

  1. Enter sudo -i to enter root privileges, you can choose to upgrade the kernel, yum update -y upgrades the kernel
  2. yum install wget download wget tool
    wget https://github.com/txthinking/brook/releases/download/v20190205/brook download brook server version
  3. Enter cp brook /usr/bin to copy brook to the bin directory
  4. chmod +x /usr/bin/brook gives the current path brook executable permissions
  5. brook -h #View help
  6. nohup ./brook server -l :80 -p pass1234 & add brook port and password and let brook run in the background

Brook settings start with the system

vi /etc/systemd/system/brook.service Create brook.service startup script with vi
To
Press “i” to enter insert mode, enter the script as follows

Press Esc, enter “shift+:”, enter wq, press Enter, save and exit

Give executable attributes and take effect:

Open brook.service service

reboot restart vps

Check if brook service is running normally

The following code:

systemctl list-unit-files Check whether the Brook script takes effect. After re-modifying the Brook password, we need to reload the information and enter systemctl daemon-reload or restart vps and enter reboot

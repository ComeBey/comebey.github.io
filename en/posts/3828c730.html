Set server login password

 2020-07-26  2020-09-20
  -Linux -centos7
  a minute read (About 142 words) 0 visits

centos7

After getting root privileges, modify the SSH configuration file /etc/ssh/sshd_config

Set a password, enter it twice, enter do not prompt any characters

Restart the SSH service, and finally use the account as root and the password for the third-party SSH login that you just set.

Debian 9 and 10

Restart the SSH service, and finally use the account as root and the password for the third-party SSH login that you just set.
